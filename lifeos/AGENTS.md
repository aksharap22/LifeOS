# AGENTS.md

## Project Name

LifeOS - Personal Optimization Engine

## Agent Architecture

### Experiment Agent

Responsibilities:

* Create experiments
* Manage experiment duration
* Track experiment progress
* Validate experiment completion

Inputs:

* Experiment Name
* Duration
* Metrics
* Hypothesis

Outputs:

* Experiment Plans
* Progress Status

---

### Data Collection Agent

Responsibilities:

* Collect user logs
* Store daily metrics
* Maintain experiment records

Inputs:

* Mood Scores
* Energy Levels
* Productivity Scores
* Sleep Data
* Exercise Data

Outputs:

* Structured Datasets

---

### Analysis Agent

Responsibilities:

* Compare experiment phases
* Identify patterns
* Evaluate hypotheses
* Generate statistical summaries

Inputs:

* Experiment Data
* User Logs

Outputs:

* Insights
* Performance Reports

---

### Recommendation Agent

Responsibilities:

* Suggest future experiments
* Recommend lifestyle improvements
* Generate personalized advice

Inputs:

* Historical Data
* Completed Experiments

Outputs:

* Recommendations
* Optimization Suggestions

---

### Life Profile Agent

Responsibilities:

* Generate user profile
* Build personal operating manual
* Maintain long-term behavioral patterns

Inputs:

* Experiment Outcomes
* User Metrics

Outputs:

* Personal Optimization Report

## Workflow

User Creates Experiment
↓
Data Collection Agent
↓
Analysis Agent
↓
Recommendation Agent
↓
Life Profile Agent
↓
Insights Dashboard

## Success Metrics

* Experiments Completed
* User Retention
* Recommendation Accuracy
* User Improvement Scores
* Daily Active Users
